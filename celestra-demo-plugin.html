This is a Celestra demo plugin. The added function is the addNumbers().

Tested on desktop browsers (latest Chrome, latest Firefox, latest Edge, IE11) and mobile devices (iOS Safari, Chrome, Firefox and Android Chrome, Samsung Internet, Firefox, Edge and W10M Edge).

Latest version: 3.5.0

Download: celestra-demo-plugin.js

Source of the plugin


/**
 * @name Celestra demo plugin
 * @version 3.5.0
 * @see https://github.com/Serrin/Celestra
 * @license MIT https://opensource.org/licenses/MIT
 * Minimal required Celestra version: 3.5.0
 */

(function(celestra){
"use strict";

celestra.addNumbers = function () {
  if (arguments.length>0) {
    var sum = 0;
    for (var i=0, l=arguments.length; i<l; i++) {
      if ( celestra.getType(arguments[i], "number") ) {
        sum += arguments[i];
      } else {
        throw "celestra.addNumbers() error: all parameters have to be number - "
          + JSON.stringify(arguments[i]);
      }
    }
    return sum;
  } else {
    throw "celestra.addNumbers() error: no parameters";
  }
};

}(celestra));

Functions

Name Description
addNumbers() Make a sum from the numbers in the parameter values. The return value is a the sum. The function throwns an error, if there is no parameter or the type of a parameter isn't a number.

Samples

addNumbers() valid values 
alert( _.addNumbers(3) );
alert( _.addNumbers(3, 5) );
alert( _.addNumbers(3.14, -5) );
addNumbers() with erros 
alert( _.addNumbers() );
alert( _.addNumbers(3, "string") );

You can check the error messages in the browser console.
